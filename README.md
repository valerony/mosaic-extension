# Mosaic Extension

Chrome расширение для автоматизации работы с платформой Mosaic. Позволяет быстро загружать данные из коллекций, импортировать записи и автоматизировать рутинные операции.

## 🎯 Основные возможности

- **Автоматическая загрузка данных** — автоматически определяет поддерживаемый сайт и загружает конфигурацию
- **Поддержка вебхуков** — интеграция с N8N для автоматизации процессов
- **Импорт CSV данных** — загрузка данных из CSV файлов в коллекции Mosaic
- **Создание записей** — массовое создание записей с поддержкой различных типов данных (текст, дата, изображения, ссылки и т.д.)
- **Кэширование** — локальное сохранение конфигурации для быстрого доступа
- **Поддержка множества сайтов** — встроенная поддержка более 13 популярных Mosaic сайтов

## 🚀 Установка

### Для разработки

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/valerony/mosaic-extension.git
cd mosaic-extension
```

2. **Откройте расширение в Chrome:**
   - Откройте `chrome://extensions/`
   - Включите "Режим разработчика" (Developer Mode)
   - Нажмите "Загрузить расширение" (Load unpacked)
   - Выберите папку проекта

## 📖 Использование

### Автоматическая загрузка

1. Откройте одну из поддерживаемых страниц
2. Нажмите на иконку расширения
3. Расширение автоматически определит сайт и загрузит данные коллекций

### Импорт CSV данных

1. Подготовьте CSV файл с данными
2. В попапе расширения выберите опцию импорта
3. Загрузите CSV файл
4. Выберите коллекцию для импорта
5. Нажмите "Создать" для массовой загрузки записей

### Использование вебхуков

1. Введите URL вебхука в поле (автоматически заполняется для поддерживаемых сайтов)
2. Нажмите на кнопку отправки вебхука
3. Данные из вебхука автоматически загружаются в попап

## 🔧 Технологии

- **Chrome API** — взаимодействие с браузером
- **Vanilla JavaScript (ES6+)** — основной язык
- **Bootstrap 5** — стилизация UI
- **PapaParse** — парсинг CSV файлов
- **N8N** — вебхуки для автоматизации

## 💾 Хранение данных

Расширение использует:
- **chrome.storage.session** — временное хранилище конфигурации (очищается при закрытии браузера)
- **sessionStorage** — данные попапа во время сессии

## 🔐 Безопасность

- Все запросы выполняются в контексте браузера
- Конфигурация хранится локально
- Поддерживаются CORS запросы к API Mosaic

## 📝 Разработка

### Добавление поддержки нового сайта

В `src/popup/ui.js` добавьте запись в `WEBHOOK_MAPPING`:

```javascript
const WEBHOOK_MAPPING = {
    // ... существующие сайты
    'новый-сайт.ru': 'https://ваш-вебхук-url.com/webhook/id'
};
```

### Изменение логики

- **ui.js** — главный файл логики попапа
- **fetcher.js** — работа с API запросами
- **render.js** — отрисовка данных на странице
